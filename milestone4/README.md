FitPlan AI  
Personalized Fitness Plan Generator  
Overview  

FitPlan AI is an intelligent web-based application that generates personalized workout and nutrition plans using AI.  
It takes user-specific inputs such as age, height, weight, fitness goals, and available equipment to create customized fitness routines and diet plans.  

Features  
User Authentication with OTP verification  
User Profile Management   
Automatic BMI Calculation  
Personalized Workout Plan Generation  
Customized Nutrition Plan  
Structured Daily/Weekly Plans  
Interactive Dashboard  
Secure Password Hashing  
How It Works:  
User registers and logs in  
Enters personal fitness details  
System calculates BMI  
AI processes user data  
Generates:  
Workout plan (sets, reps, exercises)  
Nutrition plan (daily meals)  
Displays results on dashboard  
System Architecture:  
Frontend: Streamlit + Custom CSS  
Backend: Python  
Database: SQLite  
Authentication: JWT + OTP (SendGrid API)  
Data Processing: Pandas  
Glimpse:  
Login & Signup Page  
User Dashboard  
Workout Plan Page  
Nutrition Plan Page  
Known Issues:  
Login module may face temporary issues due to OTP/email service integration  
Minor UI improvements are ongoing  
Future Scope:  
    Mobile Application  
    Integration with Wearable Devices  
    Real-time Progress Tracking  
    AI-based Calorie Tracking (Image Recognition)  
    Voice Assistant for Fitness Guidance  
    Smart Gym Equipment Integration  
Conclusion:  
FitPlan AI simplifies fitness planning by providing accessible, affordable, and personalized guidance.  
It combines AI-driven workout routines and nutrition planning into a single platform for better health outcomes.
